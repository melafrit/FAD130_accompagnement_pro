# FAD130 — Accompagner le parcours de formation et de transition professionnelle

Livrables de l'unité d'enseignement **FAD130** (Cnam) — *Accompagner le parcours de formation et de transition professionnelle*.

- **Auteur :** Mohamed Amine El Afrit
- **Intervenante :** Samia Chementel

## Objet

Production des livrables d'évaluation de l'UE à partir d'une situation réelle d'accompagnement :
**l'accompagnement d'étudiants en alternance dans la production de leur mémoire professionnel de fin d'études**
(transition du monde académique au monde professionnel).

## Livrables

| Livrable | Description | Statut |
|---|---|---|
| Grille d'auto-évaluation | Grille en 3 critères : relation d'accompagnement / mise en œuvre / positionnement professionnel | V1 (fond) |
| Support de l'oral | Réponses aux 3 questions (place & fonctions / stratégies & méthodes / rôle social) | à venir |
| Auto-évaluation /20 | Note auto-attribuée + commentaire justificatif | à venir |
